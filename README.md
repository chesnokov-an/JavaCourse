# Core Java for the impatient

This repository contains my solutions to the programming exercises from the book  
**"Core Java for the impatient"** by Cay S. Horstmann.

## Structure

Solutions are organized by chapter and task number:

`src/main/java/org/example/chapterXX/taskYY/`

Each task is placed in its own package (e.g., `chapter01.task01`) with a corresponding Java class.

## How to run

All examples are standard Java projects. You can open the root folder in any IDE (IntelliJ, Eclipse) or compile and run via command line using Maven.
